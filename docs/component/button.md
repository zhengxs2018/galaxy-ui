## API

### Button Props

| 属性 | 默认值    | 类型   | 描述               |
| ---- | --------- | ------ | ------------------ |
| as   | 'button'  | string | 元素标签或组件对象 |
| type | 'default' | string | 组件类型           |

### Button Events

| 事件  | 描述     |
| ----- | -------- |
| click | 点击事件 |
